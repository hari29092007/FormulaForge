# FormulaForge

FormulaForge is a modern web application designed to help students, engineers, and science enthusiasts quickly access important formulas across multiple disciplines. Instead of searching through textbooks, notebooks, or scattered resources, users can instantly browse, search, and explore formulas in a clean and organized interface.

## 🌐 Live Demo

https://formula-forge-beta.vercel.app/

Built with React, Vite, KaTeX, and deployed on Vercel

---

## 📖 About the Project

FormulaForge was developed as a personal project to create a centralized platform for learning and revising scientific and engineering formulas.

As a Robotics and Automation Engineering student at MIT Campus, Anna University, I wanted to build a tool that makes formula reference faster, more organized, and more accessible for students.

The platform currently contains 75 carefully selected formulas across five major disciplines:

- Mathematics
- Physics
- Chemistry
- Electrical & Electronics
- Quantum Physics

Each formula is displayed using proper mathematical rendering and organized in a clean, searchable interface to improve the learning experience.

## 🎯 Project Goal

The goal of FormulaForge is to make formula revision faster and more accessible for students. Rather than memorizing where formulas are located across multiple textbooks and notes, users can quickly search and find what they need from a single platform.
